# DIDYMOS CesiumJS visualisation app

This demo visualisation application is based on the CesiumJS library (https://cesium.com/platform/cesiumjs/). 

## Run Application

The application is a simple HTML webpage with Javascript scripts. It can be run in any local web server like Tomcat, Jetty, nginx. Any extra setup is needed.

## Data

The application contains demo data in form of JSON files for selected period 2024-07-18T13:59:00Z - 2024-07-18T14:08:00Z
